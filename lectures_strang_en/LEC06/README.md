# Lecture 06 – Column Space and Subspaces

## Objective
Show how linear combinations of two vectors generate a plane.

## Methodology
This material uses mathematical visualization to support the understanding of linear algebra concepts.

## Main Animation
- scene_plane_through_origin.py  
- plane_span_scene.mp4  

## Key Idea
\( P = \text{span}\{v, w\} \) is a plane that passes through the origin → subspace.

## Classroom Use
1. Show the animation  
2. Pause at the plane  
3. Explain linear combination  
4. Connect with the definition of subspace
